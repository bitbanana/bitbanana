## Cryptocurrency Simulation

- TypeScript on Deno
- Blockchain
- Proof of Stake

## Run

- install denon

```
$ deno install -qAf --unstable https://deno.land/x/denon/denon.ts
```

- reset all data

```
$ denon reset
```

- run main.ts

```
$ denon main
```
